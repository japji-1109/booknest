# booknest

📚 BookNest – Online Bookstore Landing Page

A modern and aesthetically pleasing bookstore landing page built using HTML5 and CSS3.
BookNest is designed to provide a cozy and elegant interface for browsing books and engaging with a reading community.

📌 Project Overview

BookNest is a static front-end website that includes:

A navigation bar with search functionality

A welcoming hero section with quote

Featured books section

Cozy reading corners showcase

Community engagement section

Upcoming events panel

Responsive design for different screen sizes

This project demonstrates strong understanding of layout design, styling, and responsiveness using CSS.

🚀 Features

✅ Clean and aesthetic beige theme

✅ Responsive navigation bar with search input

✅ Hero section with background image + gradient overlay

✅ Featured Books card layout

✅ Reading Corners section with image cards

✅ Community engagement section

✅ Upcoming Events panel

✅ Footer section

✅ Fully responsive design
